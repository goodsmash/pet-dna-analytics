# 🧬 Pet DNA AI - Quick Start (1-Minute Read)

## What Is This?

**Upload your pet's DNA file → Get instant genetic insights about breed, health, and behavior**

Think "23andMe for pets" — but you can use DNA from ANY provider (Embark, Wisdom Panel, etc.)

---

## ⚡ 3 Simple Steps

### 1. Set Up Your Pet Profile (2 min)
- Add pet name, species (dog/cat), basic info
- Choose FREE, PRO ($99/yr), or ULTRA ($199/yr)

### 2. Upload DNA File (30 sec)
- Got a DNA test from Embark/Wisdom Panel? Download the raw data file
- Tap "Select DNA File" in the app
- Upload the `.zip`, `.csv`, or `.vcf` file

### 3. View Results (instant)
- See breed breakdown ("60% Lab, 30% Golden, 10% Poodle")
- Health risks (hip dysplasia, allergies, cancer markers)
- Behavior traits (energy, trainability, sociability)
- Diet recommendations (protein needs, food sensitivities)
- **And 40+ more categories** (depending on tier)

## ❓ FAQs

**Do you sell DNA tests?**
No! Use any provider (Embark, Wisdom Panel, etc.) and upload here.

**How long does analysis take?**
30-60 seconds (or 2-3 min for ULTRA's triple-validation).

**Is this legit science?**
Yes — 91M+ genetic variants from NCBI (National Center for Biotechnology), peer-reviewed research.

**Can I cancel?**
Anytime. Your reports stay accessible forever.

**Is this a vet replacement?**
No! Use this to *inform* vet visits, not replace them.
